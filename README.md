# Health Tracker

A personal, mobile-first health tracking web application designed to identify patterns between gut health, hormone cycles, diet, and exercise.

## Goals
- Track daily health-related data in a simple, non-overwhelming way
- Identify trends over time
- Learn full-stack software engineering fundamentals

## Tech Stack (Planned)
- Frontend: React
- Backend: Node.js + Express
- Database: SQLite
